# Data Structures Implementation in C++

## Objective

Implement core data structures from scratch and demonstrate their functionality.

## Implemented Data Structures

### 1. Linked List

Operations:

* Insert
* Delete
* Search
* Display

Time Complexity:

* Insert: O(1)
* Delete: O(n)
* Search: O(n)

### 2. Stack

Operations:

* Push
* Pop
* Search
* Display

Time Complexity:

* Push: O(1)
* Pop: O(1)
* Search: O(n)

## Language Used

C++

## Files

* linked_list.cpp
* stack.cpp
